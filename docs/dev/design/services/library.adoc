= The core library

The core library is responsible for providing the rs-server functionalities.
It is independent of the API layer.
The business logic is kept independent of technical things like transfer protocol.
It is composed of shared and specific elements divided in functional modules.

Shared elements are provided by the common python project in the library package.

Some functionalities are specific to a service or add specialization to shared elements.
In this case, the functionality is provided specially for this service.
