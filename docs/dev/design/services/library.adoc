= The core library

The core library is responsible for providing the rs-server functionalities.
It is independent of the API layer.
The business logic is kept independent of technical stuff like transfer protocol.
It is divided on multiple functional modules like data_retrieval.

There is common mechanisms that are shared.
The shared elements are provided by the library package of the common python project.
